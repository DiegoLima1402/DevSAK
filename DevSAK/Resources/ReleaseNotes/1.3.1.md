**DevSAK - Atualizações Recentes - Versão 1.3.1**

**AppZip**
- Removida completamente a integração experimental com File.Kiwi.
- Removidos botão, modal, serviço, modelo, logs, links e documentação relacionados ao upload para Kiwi.
- Diálogo final do ZIP voltou ao fluxo simples:
  - Abrir Arquivo
  - Abrir Pasta
  - Fechar
- Build validado após remoção.

**MySQL Restore**
- Refatorado o pipeline de restauração para melhorar responsividade em imports grandes.
- Mantido o streaming do `.sql` para o `mysql.exe` sem carregar o arquivo inteiro em memória.
- Melhorado o envio assíncrono para o processo com buffers maiores e leitura sequencial.
- Logs agora são enviados para a UI com flush controlado, reduzindo travamentos do console em operações longas.
- Adicionado limite de log visível para preservar a responsividade.
- Melhorado tratamento de cancelamento e erros técnicos no log.
- Verificado que não há chamadas bloqueantes como `.Wait()`, `.Result`, `ReadToEnd` ou `WaitForExit()` no fluxo MySQL ajustado.

**MySQL Backup**
- Adicionada opção **“Compactar backup em ZIP”** na aba de backup.
- Quando desmarcada, o backup continua gerando `.sql` normalmente.
- Quando marcada:
  - gera o backup `.sql`;
  - compacta em `.zip` usando APIs nativas do .NET;
  - mantém o `.sql` dentro do ZIP com o mesmo nome;
  - remove o `.sql` solto após compactação;
  - mostra o caminho final como `.zip`;
  - respeita a confirmação existente de overwrite para o arquivo final.
- Adicionadas mensagens de status/log:
  - `Compactando backup em ZIP...`
  - `Backup compactado com sucesso.`

**Tema da Aplicação**
- O DevSAK agora salva automaticamente o tema selecionado pelo usuário.
- Ao reiniciar o aplicativo, o tema anteriormente escolhido (Light ou Dark) é restaurado automaticamente.
- Melhorada a persistência visual da aplicação para manter a experiência consistente entre execuções.
